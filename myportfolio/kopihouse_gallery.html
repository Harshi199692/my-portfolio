KopiHouse — Website Design

A single-page responsive website built for a local café brand. Designed with a warm dark theme, smooth navigation, and conversion-focused layout. Each section was structured to guide the user naturally from discovery to action.

UI/UX HTML / CSS / JS Responsive Single Page
01
Section 01

Hero / Landing Section

The top section of the page designed to immediately capture attention and set the brand mood.

  • Fixed navigation bar with logo, nav links, and a highlighted CTA button on the right
  • Full-viewport background image with a dark overlay for text contrast
  • Large display heading using mixed font weights and italic style for emphasis
  • Short supporting tagline below the heading
  • Two side-by-side action buttons — primary (solid) and secondary (outline)
  • Animated scroll indicator at the bottom center
  • Floating WhatsApp icon fixed to the bottom-right corner
02
Section 02

About / Brand Story Section

A two-column layout used to introduce the brand with imagery and text side by side.

  • Left column: large primary image with a smaller overlapping secondary image (stacked/floating effect)
  • Right column: section heading, two paragraphs of body text, and feature highlights
  • Feature highlights structured as icon + bold label + short description rows
  • CTA button placed at the bottom of the text column
  • Light background to contrast with the dark hero above
03
Section 03

Menu / Product Grid Section

A filterable product grid that lets users browse items by category without leaving the page.

  • Section heading and subheading centred at the top
  • Filter tab bar with category buttons (All, Hot, Cold, Food, Specials)
  • Responsive 4-column card grid — each card has an image, badge label, title, description, price, and order button
  • Distinct badge styles to highlight popular or new items
  • Dark background to visually separate this section from others
04
Section 04

Gallery / Image Grid Section

A visual-only section using an asymmetric grid to display atmosphere and lifestyle photography.

  • Centred section label and heading
  • Two-row asymmetric image grid — top row has one wide image and two smaller ones, bottom row has three equal columns
  • All images use consistent border-radius for a cohesive look
  • Light cream background to let the images breathe
  • No text inside the grid — purely visual storytelling
05
Section 05

Promotional / Offer Banner Section

A high-conversion promotional section designed to highlight a time-limited deal and push users to act.

  • Two-column layout — left: offer text and CTA, right: product image
  • Small label tag at the top to mark it as a limited-time offer
  • Large bold heading with mixed font styling for emphasis
  • Live countdown timer displayed as three separate boxes (Hours / Mins / Secs)
  • Single prominent CTA button linked to WhatsApp
  • Dark background for strong contrast and urgency
06
Section 06

Contact / Location Section

An information-focused section that gives users everything they need to visit or get in touch.

  • Two-column card layout — left card: contact details, right card: map/directions
  • Contact card uses icon + label + value rows for Location, Hours, Phone, and Email
  • Social media icon row at the bottom of the contact card
  • Right card styled as a dark panel with a location pin icon, address, and Get Directions button
  • Light page background to keep the section clean and readable
07
Section 07

Newsletter & Footer

The closing section of the page — a newsletter signup banner followed by a structured footer.

  • Newsletter banner: centred heading, subtext, inline email input + submit button, and a privacy note below
  • Contrasting background color (sage green) to visually separate this from the footer
  • Footer divided into 4 columns — logo/description, navigation links, menu links, and contact info
  • Social media icons row under the logo column
  • Bottom bar with copyright text on the left and policy links on the right
  • Dark background footer for strong page closure
